# 🏠 Home Lab & Home Server

Personal infrastructure projects built to practice real sysadmin and DevOps skills.

---

## 📦 Projects

### 🖥️ [VirtualBox Lab](./virtualbox-lab/)
Virtual home lab built on VirtualBox to practice networking and Linux administration.

**Stack:** Ubuntu Server, Docker, Samba, Uptime Kuma, SSH

---

### 🖧 [Home Server](./home-server/)
Real home server built on old desktop PC. Self-hosted cloud storage with remote access from anywhere in the world.

**Stack:** Ubuntu Server, Docker, Nextcloud, Pi-hole, Cloudflare Tunnel, ext4
